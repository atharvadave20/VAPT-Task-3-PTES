This penetration test assessed the security posture of a vulnerable Linux-based system (Metasploitable2) as part of an internal VAPT exercise. The assessment identified multiple critical weaknesses, including web application vulnerabilities, insecure service configurations, and improper privilege management. An attacker was able to chain vulnerabilities starting from a web-based attack vector to achieve remote code execution and subsequently escalate privileges to full root access. Successful exploitation demonstrated that an unauthenticated attacker could gain complete control of the system, access sensitive files, and monitor network activity. These findings highlight a high business risk due to the possibility of data compromise, service disruption, and complete system takeover. Immediate remediation actions are strongly recommended to reduce the attack surface and enforce proper security controls.
